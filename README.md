## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Debora-Carvalho/sistema-gerenciamento-psicologia.git
```

Entre no diretório do projeto

```bash
  cd sistema-gerenciamento-psicologia
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```
